tree-test-publisher — dev-only, Evolution-TZ Э2

Не часть продукта (см. CLAUDE.md: вещание — только из нативного клиента). Существует только чтобы проверить приём H.264 браузером-зрителем из дерева, пока натива (Э5) нет. Работает как «нативный» узел дерева (native:true) — публикует камеру как H.264 и ретранслирует её напрямую своим детям (assign-child).
Открывай на http://localhost:3000/dev/tree-test-publisher.html (localhost = secure context, getUserMedia работает без HTTPS).
